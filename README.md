### 精选项目

#### 🔍 [text-diff-tool](https://github.com/Dreamtale235/text-diff-tool) — 在线文本对比工具

纯前端文本对比工具，无需后端。支持按行/按字双模式对比、拖拽上传文件、DOCX/Excel 内容解析，即开即用。

#### 🎂 [qq-friends-birthday-export](https://github.com/Dreamtale235/qq-friends-birthday-export) — QQ 好友生日导出工具

基于 Playwright 的 QQ 邮箱好友生日自动爬取工具。扫码登录、遍历 12 个月日历、导出含星座和倒计时的 CSV 文件，支持 GUI 和命令行双模式。
