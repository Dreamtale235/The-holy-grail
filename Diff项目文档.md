# 文本对比工具（Diff Checker）项目文档

> 版本：v1.0 | 类型：纯前端单页工具 | 技术栈：HTML + CSS + 原生 JS

---

## 一、项目定位

一个即开即用的在线文本对比工具。用户粘贴两段文本，立刻看到差异高亮显示。

**目标用户**：程序员对比代码版本、编辑对比文稿修改、学生对比论文草稿、任何人对比两段文字差异。

**一句话描述**：粘贴左边，粘贴右边，哪里不一样一眼看出来。

---

## 二、功能清单

### v1.0 核心功能（本次实现）

| 序号 | 功能 | 描述 |
|------|------|------|
| F1 | 双栏文本输入 | 左右两个文本框，分别粘贴原始文本和修改后文本 |
| F2 | 双模式对比 | 「按行」模式逐行高亮，适合代码对比；「按字」模式逐字符高亮，适合文稿校对 |
| F3 | 按行差异高亮 | 点击对比后，新增行绿色背景、删除行红色背景、相同行无背景 |
| F4 | 按字差异高亮 | 字符级 LCS 算法 + 跨行识别，插入一个字不会导致后续行全部高亮 |
| F5 | 行号显示 | 左右两侧分别显示行号 |
| F6 | 同步滚动 | 左右两个结果面板同步滚动 |
| F7 | 一键清空 | 清空两边内容重新开始 |
| F8 | 交换左右 | 一键交换左右两边内容 |
| F9 | 差异统计 | 显示新增/删除/未修改行数统计 |
| F10 | 复制结果 | 一键复制对比结果到剪贴板 |

### v1.1 计划功能（下次迭代）

| 序号 | 功能 | 描述 |
|------|------|------|
| F11 | 忽略空白 | 切换开关：比较时忽略空格和制表符差异 |
| F12 | 忽略大小写 | 切换开关 |
| F13 | 导出结果 | 下载对比结果为 HTML/Markdown |
| F14 | 拖入文件 | 支持拖入 .txt 文件自动加载 |

## 四、技术方案

| 项目 | 选择 |
|------|------|
| 结构 | 单 HTML 文件（含内联 CSS + JS） |
| 差异算法 | 基于 LCS（最长公共子序列）的逐行比较 |
| 数据存储 | 无持久化需求（不涉及 localStorage） |
| 图表 | 不需要 |
| 外部依赖 | 零 |
| 文件大小 | 目标 < 20KB |

## 六、验收标准

- [ ] 粘贴两段不同文本，点击对比，正确标出新增行（绿）和删除行（红）
- [ ] 粘贴两段完全相同文本，显示「内容完全相同」
- [ ] 左边或右边为空时，给出合理提示
- [ ] 左右面板同步滚动
- [ ] 移动端布局正常（文本框上下排，不并排）
- [ ] 复制结果功能可用
- [ ] 无控制台报错
